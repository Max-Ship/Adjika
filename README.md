# ☕ Landing Page: Coffee Bar "Adjika" (Commercial Project)

Реализация современного адаптивного лендинга для кофейни. Проект выполнен с акцентом на типографику, плавные интерфейсные решения и кроссбраузерную стабильность.

---

## 🛠 Стек технологий и инструменты
*   **Markup:** HTML5, Семантическая верстка.
*   **Styling:** CSS3 (Custom Properties), использование **Normalize.css** для идентичного отображения во всех браузерах.
*   **Slider/UX:** Интеграция библиотеки **Swiper.js** для создания отзывчивых слайдеров.
*   **Typography:** Кастомное подключение шрифтов (MAK, TildaSans, BonaNova) через `@font-face` с оптимизацией загрузки (`font-display: swap`).
*   **Architecture:** Методология **BEM (БЭМ)**.
*   **Build Tool:** Vite (сверхбыстрая сборка и Hot Module Replacement).

---

## 🚀 Ключевые особенности
- **Performance:** Оптимизированная загрузка шрифтов и ресурсов (использование современных форматов .woff).
- **Responsive Design:** Полная адаптивность под Desktop, Tablet и Mobile устройства.
- **Detailed UI:** Проработка интерактивных элементов и плавных переходов между секциями.
- **Scalability:** Чистая структура файлов, готовая к дальнейшему расширению функционала.

---

## 💡 История проекта
Разработан для реального заказчика (кофейня "Adjika"). Полностью готов к продакшену, но переведен в демо-статус из-за закрытия бизнеса. Демонстрирует навыки коммерческой фронтенд-разработки.
